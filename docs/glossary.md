# Glossary

Contribution

A real-world action that creates value for another participant.

Node

A participant running protocol infrastructure.

Peer

A participant in the network.

Marketplace

An implementation built on top of the protocol.

Request

A request for products or services.

Offer

A proposal to provide products or services.

Settlement

The process of completing an exchange.

Identity

How participants identify themselves.

Reputation

A possible future mechanism describing historical behavior.

Credit (TBD)

Possible contribution settlement unit.

Not yet defined.
