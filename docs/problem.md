# Problem Statement

Modern digital platforms have created enormous convenience.

They have also concentrated decision-making power.

Common concerns include:

- opaque algorithmic dispatch
- high commissions
- unpredictable income
- ranking manipulation
- customer lock-in
- unilateral account suspension
- dependence on a single platform
- limited transparency

These issues appear across many industries including:

- ride-hailing
- food delivery
- local services
- local commerce

This project asks whether coordination can happen through an open protocol rather than centralized platforms.
