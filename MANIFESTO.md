# Manifesto

Technology should empower contributors, not extract from them.

Platforms should compete by building better software, not by owning the market.

Protocols should outlive companies.

People who provide real products and services should have meaningful influence over the systems they depend on.

Algorithms should be transparent whenever possible.

Communities should be able to govern their own marketplaces.

Anyone should be free to build compatible software.

Anyone should be free to run a node.

No single organization should be required for the network to exist.

This project begins with questions instead of answers.

We are not building another platform.

We are trying to make platforms optional.
