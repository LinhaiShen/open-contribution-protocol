# Principles

## Protocol before Platform

Build an open protocol instead of a company-controlled platform.

## Open before Closed

Specifications should be public.

## Contribution before Capital

The protocol should reward real-world contribution rather than financial power.

## Permissionless Participation

Anyone should be able to participate.

## Self-hosted Nodes

Anyone should be able to run infrastructure.

## Community Governance

Protocol evolution should happen through open discussion.

## Interoperability

Multiple independent implementations should coexist.

## Transparency

Rules should be understandable and inspectable.

## No Vendor Lock-in

Users should never depend on one official application.

## Experimentation

No idea is considered final until tested.
