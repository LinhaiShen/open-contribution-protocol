# Contributing

Thank you for participating.

This project is currently exploring ideas rather than implementing a finished system.

We welcome:

- protocol proposals
- architecture ideas
- economic models
- governance models
- anti-abuse mechanisms
- legal analysis
- security reviews
- UX concepts
- real-world experiments

Please avoid discussions focused on speculation or token price.

Every proposal should explain:

- the problem
- the proposed solution
- benefits
- risks
- possible abuse
- open questions

Respect different opinions.
